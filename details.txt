# 1. Project Overview

## Project Name

Grand Egyptian Museum Tourist Guide

## Project Type

AI-Powered Full-Stack Museum Guide Platform

## Main Goal

The project is designed to provide tourists and museum visitors with an intelligent digital assistant for exploring ancient Egyptian monuments and artifacts.

The platform combines:

* Computer Vision
* Retrieval-Augmented Generation (RAG)
* REST APIs
* Microservices Architecture
* Multilingual Museum Content
* AI-Powered Historical Assistance

The system allows users to:

* Scan Egyptian artifacts using AI image recognition
* Receive monument information in multiple languages
* Ask AI questions about ancient Egyptian history
* Generate monument descriptions using AI
* Save scans and gallery items
* Manage user profiles
* Add monument reviews
* Access personalized scan history

* Python

* Receive uploaded images
* Predict artifact classes
* Return confidence scores

## Main Endpoint

```http

```

## AI Flow

```text
Image Upload
    ↓
Backend
    ↓
ai-service /predict
    ↓
TensorFlow model inference
    ↓
```

# 14. Artifact Recognition Detailed Flow

```text
User uploads image
        ↓
Frontend sends multipart request
        ↓
Backend receives image
        ↓
Multer processes upload
        ↓
Backend forwards image to ai-service
        ↓
TensorFlow predicts class
        ↓
Backend maps AI class to monument
        ↓
Scan session saved in PostgreSQL
        ↓
Backend calls RAG /describe
        ↓
AI historical description generated
        ↓
Complete response returned
```

---

# 15. RAG Historical Q&A Flow

```text
User asks question
        ↓
Frontend sends question to backend
        ↓
Backend forwards request to rag-service
        ↓
ChromaDB semantic retrieval
        ↓
Relevant chunks retrieved
        ↓
Qwen 2.5 generates answer
        ↓
Backend returns response
        ↓
Frontend displays answer
```

# 16. Forgot Password Flow

```text
User enters email
        ↓

        ↓
OTP hashed and stored
        ↓
OTP sent by email
        ↓
User enters OTP
        ↓
Backend verifies OTP
        ↓
User enters new password
        ↓
Password updated
        ↓
OTP cleared
```

# 18. AI Classes Mapping

AI class names are mapped from:

```text
ai-service/model/class_names.json
```

The backend mapping file:

```text
backend/utils/classMapping.js
```

Known unmapped class:

```text
Amenhotep_III_Tiye
```

If predicted, backend returns:

```text
not_found
```
